## Next Tiptap

A modern and feature-rich Rich Text Editor template, built with Next.js 14 and TipTap.

[Demo](https://next-tiptap.vercel.app/)

### Features

- 📝 Rich Text Editor + HTML Mode
- 🌓 Light/Dark themes
- 📱 Responsive design
- 📊 Content Rendering (SSR/CSR)

### Tech Stack

- Next.js 14
- TipTap
- CodeMirror
- TypeScript
- SCSS + Tailwind CSS
